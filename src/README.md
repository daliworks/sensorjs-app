## Requirements

### Software
- node: v0.10.x

### Hardware
- Beaglebone Black
- Sensors(Optional)
  - Temperature: DS18B20
  - Humidity: HTU21D
  - Light: BH1750
- Actuators(Optional)
  - RGB Led

## Install

### Device App
```
cd [source-directory]
cd device-app
npm install
```

### Device Client
```
cd [source-directory]
cd device-client
bower install
npm install (optional)
```

## Getting Started
```
cd device-app
node app.js
```

__Open browser and access__
